# 🌐 Methodex – Website Reimagined

A modern and responsive version of the Methodex website — designed and developed by **Rohit Khandelwal** using React and Tailwind CSS.  
It focuses on rebranding, clean UI, and a smooth user experience.

---

## 🚀 Features

- ⚡ **Fast & Responsive** — Works perfectly on all devices.  
- 🎨 **Modern UI** — Fresh colors, fonts, and smooth animations.  
- 🧩 **Reusable Components** — Modular structure using React props.  
- 💬 **Interactive Design** — Popups, smooth scrolling, and instant form validation.  
- 🔄 **Live Deployment** — Hosted on Vercel for fast global access.

## 🧑‍💻 How to Run This Project Locally

Follow these simple steps to set up the project on your local device 👇

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/rohit2004khandelwal/methodex.git
```

### 2️⃣ Go Inside the Project Folder
```bash
cd methodex
```

### 3️⃣ Install Dependencies
Make sure you have **Node.js** installed, then run:
```bash
npm install
```

### 4️⃣ Start the Development Server
```bash
npm run dev
```

### 5️⃣ Open in Browser
Once the server starts, open this link in your browser:
```
http://localhost:5173/
```

Now you can explore and edit the project locally 🎨  
Any changes you make will automatically update in the browser.
